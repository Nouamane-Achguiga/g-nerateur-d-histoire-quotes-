Inspirational Quotes Generator
This repository contains a simple JavaScript program that generates random inspirational quotes from a pre-defined array of quotes.

Getting Started
To use this program, simply open the index.html file in a web browser. The program should appear in the browser window and you can click the "Generate Quote" button to display a random quote.

Usage
The program generates random quotes from an array of pre-defined quotes. When you click the "Generate Quote" button, the program selects a random quote from the array and displays it in the p element.

Files
This repository contains the following files:

index.html - the main HTML file containing the program markup
quotes.css - the CSS file containing the program styling
quotes.js - the JavaScript file containing the program logic

How to Use
To use the quotes generator:

Open the index.html file in a web browser.
Click the "click to change" button to generate a new random quote.
Customization
To customize the quotes:

Open the quotes.js file in a text editor.
Replace the quotes in the quotes array with your own quotes.
Save the file.
